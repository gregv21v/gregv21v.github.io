Updating the App.js File

In your apps folder, there is a file named App.js. This file contains the main entry point of the app. When you create the project there is code already present in App.js. It should look like this. 

        
          import { StatusBar } from 'expo-status-bar';
          import React from 'react';
          import { StyleSheet, Text, View } from 'react-native';

          export default function App() {
            return (
              
                Open up App.js to start working on your app!
                
              
            );
          }

          const styles = StyleSheet.create({
            container: {
              flex: 1,
              backgroundColor: '#fff',
              alignItems: 'center',
              justifyContent: 'center',
            },
          });

We are going to make changes to this code. Start by removed the StatusBar from the App Component. Remove the corresponding import as well. We won't need that. Replace the text in the text component with 'Tic Tac Toe Game'. This is just filler text. We are going toe replace the whole component with something different later. We are going to want to change the View to SafeAreaView, so that our app will be more compatible with mobile devices. Don't forget to change the import from View to SafeAreaView as well. When you are done your code should look like this: 

        
          import React from 'react';
          import { StyleSheet, Text, SafeAreaView } from 'react-native';

          export default function App() {
            return (
              <View style={styles.container}>
                <Text>Tic Tac Toe Game
              </View>
            );
          }

          const styles = StyleSheet.create({
            container: {
              flex: 1,
              backgroundColor: '#fff',
              alignItems: 'center',
              justifyContent: 'center',
            },
          });

Test your code to make sure it works
