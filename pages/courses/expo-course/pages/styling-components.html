Styling Components

There are two ways to style a component in Expo/React. The first way is inline. The second is by creating a style sheet with the StyleSheet.create function.

Inline: 

          
            <View style={{ flex: 1/3, padding: 20 }}>
            </View>

The style is enclosed in double brackets instead of single brackets because it is passing an object to the styles prop.

Global: 

          
            // begin component ...
            // begin render method
            <View style={styles.styleName}>
            </View>
            // end render method
            // ... end component

            const styles = StyleSheet.create({
              styleName: {
                fontSize: 40
              }
            })
