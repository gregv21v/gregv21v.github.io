Props

Each component has a props variable for it. The props variable is passed to the constructor. When you write the tags for a component, you pass props to the function as if they where attributes to an html tag. 


            render() {
              return (
                <ComponentName prop1Name={propValue} prop2Name="propValue" />
              )
            }

Then you can refer to the props by their name. For example, in the class ComponentName, you could refer to the prop prop1Name by this.props.prop1Name.

The child components of a Component are also a prop reference by this.props.children.

  1. In the cells component's render method use a value prop as the text in the text tags 

    
                    render() {
                      return (
                        <View>
                          <Text>{this.props.value}</Text>
                        </View>
                      )
                    }
