Introduction

This course will guide you through the process of creating a Tic Tac Toe game using the Expo SDK. You will learn how to build components, and how states, and props work. You will learn how to test your app on IOS and through the web.

What is expo?

Expo is a multi-platform SDK. You can write code with Expo then have it built on Web, Android, and IOS platforms. It's built off of React Native, so it uses a lot of react native's code.

Why use expo?

Expo is good if you want to develop a cross platform app for both Android and IOS but are working on a windows computer. You can compile your Expo code remotely with Expo builds and run it through the Expo simulator app on IOS or Android.

Need to Know

I'm assuming that you are familiar with JavaScript and HTML.
